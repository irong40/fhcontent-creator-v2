  <files>n8n-workflows/CONCURRENCY-SPIKE.md</files>
  <action>
In n8n UI, create a workflow named "SPIKE-Concurrency-Test":

1. Add **Manual Trigger** node.

2. Add **Code** node named "Create 6 Items":
```javascript
// Create 6 items to fan out to parallel branches
return [1, 2, 3, 4, 5, 6].map(i => ({
  json: { branch: i, start_time: new Date().toISOString() }
}));
```

3. Add **Split In Batches** node named "Fan Out":
   - Batch size: 1 (processes one item at a time if serial, or all at once if parallel)
   - This is the key test: does n8n process all 6 items simultaneously?

4. Add **Code** node named "Sleep 5 Seconds":
```javascript
// Simulate slow API call (e.g., HeyGen job initiation)
const branch = $input.item.json.branch;
const start = Date.now();

// Busy wait for 5 seconds (Code node has no async sleep)
// Instead, just record timing — actual sleep not possible in sync Code node
// Use Wait node for real sleep

return [{
  json: {
    branch,
    start_time: $input.item.json.start_time,
    processed_at: new Date().toISOString()
  }
}];
```

5. Add **Wait** node named "Wait 5s" after the Code node:
   - Wait amount: 5 seconds
   - Resume: After wait interval

6. Add **Code** node named "Record Completion":
```javascript
return [{
  json: {
    branch: $input.item.json.branch,
    start_time: $input.item.json.start_time,
    completed_at: new Date().toISOString()
  }
}];
```

7. Add **Merge** node to collect all 6 results.

8. Add **Code** node named "Analyze Timing":
```javascript
const results = $input.all().map(item => item.json);
results.sort((a, b) => new Date(a.start_time) - new Date(b.start_time));

const first_start = new Date(results[0].start_time);
const last_complete = new Date(results[results.length - 1].completed_at);
const total_elapsed_ms = last_complete - first_start;

// If parallel: ~5s total (all branches run simultaneously)
// If serial: ~30s total (6 branches × 5s each)
const is_parallel = total_elapsed_ms < 10000; // under 10 seconds = parallel

return [{
  json: {
    results,
    total_elapsed_ms,
    appears_parallel: is_parallel,
    verdict: is_parallel
      ? 'PARALLEL: all 6 branches ran simultaneously'
      : `SERIAL: branches ran sequentially (${total_elapsed_ms}ms total)`
  }
}];
```
